Installation Guide

Clara is available for Windows, macOS, and Linux. Choose the installation method that works best for you.

System Requirements​

Minimum Requirements​

  • OS: Windows 10+, macOS 10.15+, or Linux (Ubuntu 18.04+)
  • RAM: 4GB (8GB recommended)
  • Storage: 2GB free space
  • Node.js: Version 18+ (for development)

Recommended Requirements​

  • RAM: 16GB+ (for local AI models)
  • GPU: NVIDIA GPU with 8GB+ VRAM (for image generation)
  • Storage: 10GB+ free space (for models and data)

Download & Install​

Option 1: Pre-built Releases (Recommended)​

Download the latest release for your platform:

Windows​

  1. Download Clara-Setup-x.x.x.exe from GitHub Releases
  2. Run the installer as Administrator
  3. Follow the installation wizard
  4. Launch Clara from the Start Menu

macOS​

  1. Download Clara-x.x.x.dmg from GitHub Releases
  2. Open the DMG file
  3. Drag Clara to your Applications folder
  4. Launch Clara from Applications
macOS Security

If you see a security warning, go to System Preferences > Security & Privacy and click "Open Anyway"

Linux​

  1. Download Clara-x.x.x.AppImage from GitHub Releases
  2. Make it executable: chmod +x Clara-x.x.x.AppImage
  3. Run: ./Clara-x.x.x.AppImage

Option 2: Package Managers​

Windows (Chocolatey)​

choco install clara

macOS (Homebrew)​

brew install --cask clara

Linux (Snap)​

sudo snap install clara

Development Installation​

For developers who want to build from source:

Prerequisites​

  • Node.js 18+
  • npm or yarn
  • Git

Build from Source​

# Clone the repository
git clone https://github.com/your-org/clara.git
cd clara

# Install dependencies
npm install

# Start development server
npm run dev

# Or build for production
npm run build
npm run electron:build

AI Provider Setup​

Clara requires at least one AI provider to function. Set up your preferred provider:

OpenAI​

  1. Get an API key from OpenAI
  2. Add to Clara settings: sk-your-openai-key-here

Anthropic​

  1. Get an API key from Anthropic
  2. Add to Clara settings: sk-ant-your-anthropic-key-here

Ollama (Local)​

  1. Install Ollama: curl -fsSL https://ollama.ai/install.sh | sh
  2. Pull a model: ollama pull llama2
  3. Clara will auto-detect local Ollama

Verification​

After installation, verify Clara is working:

  1. Launch Clara
  2. Complete onboarding - Enter your name and configure AI provider
  3. Test AI Assistant - Send a message: "Hello Clara!"
  4. Check features - Try image generation and agent studio

Troubleshooting​

Common Issues​

Installation Fails​

  • Windows: Run installer as Administrator
  • macOS: Check Security & Privacy settings
  • Linux: Ensure AppImage has execute permissions

Clara Won't Start​

  • Check system requirements
  • Verify Node.js version (18+)
  • Clear application data and restart

AI Provider Issues​

  • Verify API keys are correct
  • Check internet connectivity
  • Ensure sufficient API credits

Performance Issues​

  • Close unnecessary applications
  • Increase system RAM
  • Use local models (Ollama) for better privacy

Getting Help​

If you encounter issues:

  1. Check logs - Look in Clara's debug console
  2. Search issues - Check GitHub Issues
  3. Report bugs - Create a new issue with details
  4. Community - Join our community discussions
