🚀 Quick Start Guide

Get Clara up and running in just a few minutes! This guide will walk you through the essentials to start using Clara's powerful AI-powered workspace.

Prerequisites​

Before you begin, make sure you have:

  • Node.js (version 18 or higher)
  • A modern web browser (Chrome, Firefox, Safari, Edge)
  • An AI provider API key (OpenAI, Anthropic, or local Ollama)

Installation​

Option 1: Download Release (Recommended)​

  1. Visit the Clara Releases page
  2. Download the latest version for your platform:
    • Windows: Clara-Setup-x.x.x.exe
    • macOS: Clara-x.x.x.dmg
    • Linux: Clara-x.x.x.AppImage
  3. Install and launch Clara

Option 2: Build from Source​

# Clone the repository
git clone https://github.com/your-org/clara.git
cd clara

# Install dependencies
npm install

# Start development server
npm run dev

First Launch Setup​

When you first open Clara, you'll be guided through a quick onboarding process:

1. Welcome Screen​

  • Enter your name for personalization
  • Choose your preferred theme (Light/Dark)

2. AI Provider Configuration​

Choose and configure your AI provider:

OpenAI Setup​

# Required: Your OpenAI API key
API_KEY=sk-your-openai-api-key-here

Anthropic Setup​

# Required: Your Anthropic API key  
API_KEY=sk-ant-your-anthropic-api-key-here

Ollama Setup (Local)​

# Install Ollama first
curl -fsSL https://ollama.ai/install.sh | sh

# Pull a model
ollama pull llama2

# Clara will auto-detect local Ollama

3. Verify Installation​

After setup, test your installation:

  1. Open Clara Assistant - Click the chat icon in the sidebar
  2. Send a test message - Try: "Hello Clara, tell me about your features"
  3. Check response - You should get a helpful response about Clara's capabilities

Essential First Steps​

🤖 Try the AI Assistant​

  1. Access the Assistant:

    • Click the Bot icon in the sidebar
    • Or use the keyboard shortcut Ctrl+Shift+A (Windows/Linux) or Cmd+Shift+A (macOS)

  2. Start a Conversation:

    Hello Clara! What can you help me with today?

  3. Explore Capabilities:

    • Ask questions about any topic
    • Request code assistance
    • Get help with writing and analysis
    • Use tools and integrations

🎨 Generate Your First Image​

  1. Open Image Generation:

    • Click Image in the sidebar
    • Or navigate to the Image Generation tab

  2. Create an Image:

    A serene Japanese garden with cherry blossoms in full bloom, 
    soft pink petals falling, peaceful zen atmosphere, 
    high quality, detailed

  3. Customize Settings:

    • Adjust image size and quality
    • Try different art styles
    • Experiment with prompts

⚡ Build Your First Agent​

  1. Access Agent Studio:

    • Click Agents in the sidebar
    • Start with the "Create New Agent" button

  2. Use the Visual Builder:

    • Drag and drop nodes to create workflows
    • Connect inputs and outputs
    • Configure node parameters

  3. Test Your Agent:

    • Use the built-in testing tools
    • Debug workflows step by step
    • Export and share your agents

Quick Tips for Success​

Pro Tips
  • Explore the Dashboard: Get familiar with the overview and quick actions
  • Check Settings: Customize Clara to match your workflow preferences
  • Use Keyboard Shortcuts: Speed up your workflow with built-in shortcuts
  • Save Frequently: Your work is automatically saved, but manual saves are good practice
Common Issues
  • API Keys: Make sure your AI provider API keys are valid and have sufficient credits
  • Network: Clara requires internet connectivity for cloud AI providers
  • Memory: For local models (Ollama), ensure you have sufficient RAM
